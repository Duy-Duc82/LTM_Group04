# Ai-la-triệu-phú (Who Wants to Be a Millionaire Online)

Dự án phòng lab IT4062_161265 nhằm xây dựng trò chơi đố vui nhiều người chơi theo mô hình client–server với server viết bằng C.

## Khởi đầu nhanh
1. Cài đặt trình biên dịch C, thư viện socket và Python theo yêu cầu trong tài liệu.
2. Chạy `make` trong thư mục `server-c` để biên dịch và cấu hình server sau khi hiện thực.
3. Khởi chạy client mẫu (Python, C hoặc React) để kết nối và kiểm thử chức năng.
